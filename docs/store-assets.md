# Questime Store Asset Direction

This is the working launch direction for app store assets. Treat it as a starter system that can change once the product UI settles.

## Brand Direction

Questime should feel focused, capable, and slightly game-like without looking like a casual game. The visual language should connect time, quests, proof, and coaching.

- Primary background: `#0B0F14` graphite black
- Primary surface: `#151B24` deep steel
- Primary action: `#F4B942` quest gold
- Secondary action: `#33C2FF` clear cyan
- Success: `#2ED47A` progress green
- Text: `#F5F7FA` near white

## App Icon Concept

Use a no-text app icon. The clearest first version is a stylized `Q` built from a clock ring and path marker:

- Deep graphite rounded-square background
- Quest-gold circular timer ring forming the `Q`
- Cyan path tail cutting out of the lower-right of the ring
- Small white clock hand or check mark inside the ring
- Minimal depth, high contrast, readable at small sizes

Prompt for generation:

```text
A premium mobile app icon for "Questime", no words, no letters except an abstract Q shape formed by a clock ring and quest path marker, deep graphite black rounded-square background, quest-gold circular timer ring, cyan path tail, small white check mark inside, crisp vector-like 3D polish, high contrast, centered composition, App Store ready, no mockup, no device, no text
```

## Screenshot Sequence

1. Turn time into quests
   - Show the main quest list with progress and a clear primary action.
2. Prove progress with photos
   - Show before/after proof and AI verification feedback.
3. Get Handler coaching
   - Show a Handler chat helping convert a goal into a concrete quest.
4. Build accountability
   - Show social/friend quest assignment or shared progress once that flow is stable.
5. Track completed wins
   - Show completed quests, stars, and streak-like momentum.

## Store Copy Draft

App Store subtitle:

```text
Goals as guided quests
```

Google Play short description:

```text
Turn goals into quests with AI proof, coaching, and accountability.
```

Long description draft:

```text
Questime helps you turn vague goals into clear, finishable quests. Create a quest, define the proof, upload progress photos, and get AI-assisted verification from a personalized Handler coach.

Use Questime to stay accountable, build momentum, and make your time visible. Whether you are cleaning, studying, training, practicing, or helping someone else stay on track, Questime gives each effort a concrete finish line.

Core features:
- Create quests with clear completion criteria
- Get personalized Handler coaching
- Verify progress with proof photos
- Earn star feedback after completion
- Prepare for social accountability with friend-assigned quests
```

## Required Store Assets

- App icon PNGs for Apple and Google Play
- iPhone screenshots for current required App Store sizes
- iPad screenshots only if iPad support remains enabled
- Android phone screenshots
- Google Play feature graphic
- Privacy policy URL
- Support URL and email
- AI-assisted verification disclosure
- Camera and photo usage disclosure
