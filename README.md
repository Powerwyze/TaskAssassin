# Task Assassin (Flutter)

## Setup
1. **Enable Developer Mode** on Windows:
   - Settings -> Update & Security -> For developers -> Developer Mode ON.
   - This is required for Flutter plugins to build correctly on Windows.

2. **Install Dependencies**:
   ```bash
   flutter pub get
   ```

3. **Run the App**:
   ```bash
   flutter run
   ```

## Structure
- `lib/main.dart`: Entry point.
- `lib/screens/`: Dashboard, Mission setup, Execution, Profile.
- `lib/providers/`: State management.
- `lib/models/`: Data models.
